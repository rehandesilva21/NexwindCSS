Nexwind Utility Classes Documentation

Explore the complete utility classes available in Nexwind Framework with examples.

Text Colors

Use these classes to change the text color:

.tc-r500 (Red Text)

.tc-b300 (Blue Text)

.tc-g400 (Green Text)

.tc-y500(Gray Text)

.tc-white on black background

.tc-gray on black background

Background Colors

Apply background colors with these classes:

.bg-r500
.bg-b300
.bg-g400
.bg-gray300

Padding Utilities

Predefined padding classes

.p-5 (Padding all sides 5)
.p-2 .pl-5 (Padding left 5)
.pt-3 .pb-3 .pr-4 .pl-1

Padding Utilities (Custom with data-attributes)

Use data attributes to customize padding in rem units:

data-p="1" (1rem padding)
data-pt="2" (2rem padding top)
data-px="0.5"
data-py="1.5"

Margin Utilities

Predefined margin classes:

.m-4 (Margin all sides 4)
.mt-2 .mb-5 (Margin top 2, bottom 5)
.ml-3 .mr-3 (Margin left and right 3)

Margin Utilities (Custom with data-attributes)

Customize margins using data attributes:

data-m="1.5"
data-mt="2"
data-mx="0.75"
data-my="1"

data-font-size="2.5"
